# legacy-shop-spring

This is a controlled Spring Boot legacy project generated by TraceGate Eval.

It intentionally keeps five historical constraints:

- Auth must keep `legacyToken` for old mobile clients.
- Order must keep `orderStatus` and `refundStatus` independent.
- User deletion must be soft delete with `status = 2`.
- Payment signatures must use original `amountInCent`.
- Bill sync must use `syncBatchId` idempotency.

Run:

```bash
mvn test
```
